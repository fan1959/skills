# ppt-template-creator

## 模式

Generator (meta-skill)

## 触发词

- `根据这个 PPT 模板做个 skill`
- `把这个 PPT 模板封装成可复用 skill`
- `create skill from this ppt`

## 输出

SKILL.md + assets/template.pptx

## 适用场景

把 SCU 答辩 PPT / 四川大学模板 PPT 等模板封装成可复用的 Skill

## 来源

- fork 自 [chujianyun/skills](https://github.com/chujianyun/skills)
- 基于 Anthropic 官方 skill 格式
