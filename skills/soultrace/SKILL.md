# SoulTrace

通过 5 色心理模型测量用户性格，输出原型报告。

## 设计模式

本 skill 是 **API Wrapper**：
- 调用 SoulTrace API 获取题目
- 收集用户回答
- 提交答案获取原型结果
- 格式化展示报告

## 5 色模型

| 颜色 | 维度 | 描述 |
|------|------|------|
| White | 结构 | 秩序感、计划性、规则遵循 |
| Blue | 理解 | 好奇心、分析力、抽象思维 |
| Black | 能动 | 目标导向、执行力、掌控欲 |
| Red | 强度 | 情绪强度、激情、紧迫感 |
| Green | 连接 | 共情、社交、和谐取向 |

5 维度组合产生 25 种原型。

## Gotchas

- **题目是自适应的**：下一题取决于之前的回答，不能一次全部展示
- **中途退出会丢失进度**：API 不保存中间状态，必须一次完成
- **不要解释题目**：用户对题目有疑问时只说"按你的第一反应选择"
- **原型不是标签**：交付结果时强调"这是倾向，不是定论"

## 工作流

```text
测试进度：
- [ ] 步骤 1：确认用户参与意愿
- [ ] 步骤 2：调用 API 获取第一题
- [ ] 步骤 3：逐题收集回答
- [ ] 步骤 4：提交结果获取原型
- [ ] 步骤 5：格式化展示报告
```

### 步骤 1：确认意愿

简短说明：约 15-25 题，每题 2-4 选项，3-5 分钟完成。

### 步骤 2：获取第一题

调用 SoulTrace API 开始测试会话。

### 步骤 3：逐题呈现

- 一次只显示一道题
- 用 AskUserQuestion 呈现选项
- 不透露原型倾向

### 步骤 4：获取结果

提交全部回答后，API 返回：
- 5 色分数分布
- 主要原型（1/25）
- 原型描述

### 步骤 5：展示报告

- 原型名称和一句话概括
- 5 维度雷达图（文字版）
- 优势与盲区
- 适配团队角色建议

## 输出要求

- 原型名称 + 描述
- 5 色分数（数值或百分比）
- 1-2 段个性化解读
- 不输出原始 API 响应
