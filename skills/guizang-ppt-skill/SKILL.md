# Guizang PPT Skill

把内容转化为横向翻页的网页 PPT，一个 HTML 文件即可演示。

## 设计模式

本 skill 采用 **Generator** 模式：
- 接收用户的内容大纲或 Markdown 文档
- 选择风格模板
- 生成单 HTML 文件（内嵌 CSS + JS + WebGL）

## 两种风格

| 风格 | 字体 | 背景 | 配色 |
|------|------|------|------|
| 电子杂志 × 电子墨水 | 衬线体 | WebGL 流体 | 暖色系 |
| 瑞士国际主义 | 无衬线 | CSS 网格点阵 | IKB / 柠檬黄 / 柠檬绿 / 安全橙 |

## Gotchas

- **单文件限制**：所有资源（字体、图标、图片）必须内嵌或 CDN 引用，不能依赖本地文件
- **WebGL 性能**：移动端低性能设备上 WebGL 背景可能掉帧，需提供降级方案（纯 CSS 背景）
- **翻页动画**：用 CSS `scroll-snap` 而非 JS 动画库，保证 60fps
- **图片压缩**：内嵌图片用 WebP 或 base64 压缩，单文件控制在 5MB 以内
- 不要硬编码内容——从用户提供的 Markdown/大纲动态生成

## 工作流

```text
生成进度：
- [ ] 步骤 1：收集内容和确认风格
- [ ] 步骤 2：选择模板和配色
- [ ] 步骤 3：结构化内容为章节
- [ ] 步骤 4：生成 HTML
- [ ] 步骤 5：验证渲染
```

### 步骤 1：收集内容

用户提供：
- Markdown 文档 / 大纲 / 口头描述
- 风格偏好（杂志风 / 瑞士风）
- 是否需要数据大字报、图片网格等特殊模板

### 步骤 2：选择模板

- 文化艺术/个人分享 → 杂志风（衬线 + 流体背景 + 暖色）
- 技术分享/产品发布 → 瑞士风（无衬线 + 网格 + 高亮色）

### 步骤 3：结构化

将内容拆为 5-15 个章节，每章一个 `<section>`：
- 封面（标题 + 作者 + 日期）
- 章节幕封（大号序号 + 章节名）
- 内容页（文字 + 图片/代码/数据）
- 数据大字报（大号数字 + 标签）
- 图片网格（2×2 或 3×2）
- 尾页（谢谢 / Q&A）

### 步骤 4：生成 HTML

- `scroll-snap-type: x mandatory` 实现横向翻页
- WebGL 背景用 Three.js CDN（或纯 CSS fallback）
- 响应式适配 16:9 和 4:3

### 步骤 5：验证

- 桌面端 Chrome/Edge 翻页流畅
- 移动端触摸滑动正常
- 无 JS 报错

## 输出要求

- 单个 `.html` 文件
- 可直接双击在浏览器打开
- 支持键盘左右箭头翻页
- 移动端支持触摸滑动
